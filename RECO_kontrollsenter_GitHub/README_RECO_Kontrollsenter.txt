
# RECO Kontrollsenter – README

Denne veiledningen forklarer hvordan du setter opp og bruker RECO Kontrollsenter via https://reco-kalkyle.netlify.app

---

## 🔐 Innlogging
1. Gå til https://reco-kalkyle.netlify.app
2. Logg inn med e-post (f.eks. sb@reco.no)
3. Velg rolle: Avdelingsleder, Prosjektleder eller Tømrer
4. Ved første innlogging får du:
   - Demo-prosjekt
   - Opplæringsvideo
   - Språkvalg (norsk/engelsk)

---

## 🏠 Startsiden
Startsiden inneholder:
- RECO-logo og språkvalg
- Fire hovedknapper:
  1. Start demo-prosjekt
  2. Last ned demopakke
  3. Start opplæringsvideo
  4. Gå til kalkylemoduler
- Chat-knapp: "Chat med RECO" nederst til høyre
- Footer med kontaktinfo

---

## 📚 Opplæringsside
- Automatisk video ved første innlogging
- Nedlastbar MP4
- Interaktiv veiledning per rolle
- Rolleoversikt med tilgangsnivåer
- Knapper:
  - Start demo-prosjekt
  - Last ned demopakke
  - Start opplæringsvideo
  - Last ned opplæringsvideo

---

## 🧩 Moduler
- Kalkylebygger med tiltaksgenerering
- Plantegning med statusindikatorer
- Tiltak per rom med eksport og filter
- Statuslogg per prosjekt med fremdrift
- Mobilvisning med zoom og filter
- Teams-integrasjon og e-postutsendelse

---

## 🛠️ Admin-side
- Brukeroversikt med roller og tilgangsnivåer
- Rediger roller og moduler
- Legg til ny bruker
- Oversikt over rapporter og kalkyler
- Fremdriftsindikator og eksport
- Lanseringsknapp: "Åpne RECO Kontrollsenter for hele avdelingen"

---

## 📦 Filopplasting og kalkyle
1. Gå til kalkylemodulen
2. Last opp kalkylefil (f.eks. `Kalkylebygger riktig  2 (kopi).xlsx`)
3. Systemet genererer tiltak automatisk
4. Du kan eksportere tiltak som Excel, CSV eller PDF

---

## 📁 Relevante filer
- Kalkylefil: `Kalkylebygger riktig  2 (kopi).xlsx`
- Rolleoversikt: `rolleoversikt_RECO.xlsx`
- Statuslogg: `statuslogg_per_prosjekt.xlsx`
- Tiltaksliste: `tiltaksliste_per_rom.csv`
- Publiseringsrapport: `Publiseringsrapport_RECO_Kontrollsenter.pdf`
- Demopakke: `RECO_Demopakke.zip`
